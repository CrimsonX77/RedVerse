⧫ One-Command Install

Copy and run the command for your operating system. This installs Ollama, pulls the Oracle model, installs Python dependencies, and starts the backend server.

Linux & macOS

Bash — tested on Ubuntu 22+, Debian 12+, Fedora 38+, macOS 13+
curl -fsSL https://crimsonx77.github.io/RedVerse/installers/setup_redverse.sh | bash

Or download and inspect first:

curl -fsSL -o setup_redverse.sh https://crimsonx77.github.io/RedVerse/installers/setup_redverse.sh chmod +x setup_redverse.sh ./setup_redverse.sh

Windows

PowerShell (run as Administrator)
irm https://crimsonx77.github.io/RedVerse/installers/setup_redverse.ps1 | iex

Or download and inspect first:

Invoke-WebRequest -Uri https://crimsonx77.github.io/RedVerse/installers/setup_redverse.ps1 -OutFile setup_redverse.ps1 Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass .\setup_redverse.ps1

⧫ What Gets Installed

Core Components

  1. Ollama — local LLM inference engine (ollama.com)
  2. CrimsonDragonX7/Oracle:latest — the Oracle model (~4 GB)
  3. Python 3.10+ — if not already installed
  4. pip packages: edge-tts, SpeechRecognition
  5. ffmpeg — for audio transcription
  6. RedVerse repository — cloned from GitHub

Stable Diffusion optional

Scene image generation for E-Drive — requires a decent GPU (4GB+ VRAM)

After running the main installer, optionally install SD WebUI:

curl -fsSL https://crimsonx77.github.io/RedVerse/installers/setup_sd.sh | bash
  1. Installs stable-diffusion-webui (AUTOMATIC1111) in headless/API mode
  2. Downloads anyorangemixAnything_mint checkpoint from CivitAI
  3. Configures API-only mode on port 7860

⧫ Manual Installation

If you prefer to install components individually:

Step-by-Step

  1. Install Ollama: curl -fsSL https://ollama.com/install.sh | sh
  2. Pull the Oracle model: ollama pull CrimsonDragonX7/Oracle:latest
  3. Install Python dependencies: pip install edge-tts SpeechRecognition
  4. Install ffmpeg: sudo apt install ffmpeg (or brew install ffmpeg)
  5. Clone the RedVerse repo: git clone https://github.com/CrimsonX77/RedVerse.git
  6. Start the backend: cd RedVerse && python3 serve_edrive.py
  7. Open http://localhost:8666/EDrive.html in your browser
